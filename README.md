# EdTech Learning Analytics – SQL Practice Project

## Overview

This repository contains SQL queries and analytical exercises focused on learning analytics and educational data. The project demonstrates practical SQL skills used in data analysis, reporting, and business intelligence roles.

## Skills Covered

* Data Retrieval
* Filtering & Sorting
* Aggregations
* GROUP BY & HAVING
* Joins
* Subqueries
* Window Functions
* Common Table Expressions (CTEs)
* Analytical SQL Queries

## Tools

* SQL
* MySQL / PostgreSQL Compatible Queries

## Project Structure

* Basic SQL Queries
* Intermediate SQL Analysis
* Advanced Analytical Queries
* Learning Analytics Use Cases

## Learning Outcomes

* Generating insights from structured datasets
* Preparing for Data Analyst and Business Analyst interviews
